Digital Ledger - Day 3

A client-side digital ledger built with HTML, CSS, and JS. Track income/expenses and visualize your balance over time with a clean blue theme.  

Features:
- Add, edit, delete transactions (description, amount, type, date, category)  
- Filter by type and date  
- Summary: balance, total income, total expense  
- Interactive balance-over-time graph using Chart.js  
- Persistent data using localStorage  
- Responsive design with classy matte blue theme  

How to Use:
1. Open `index.html` in a browser  
2. Add transactions in the Add Transaction tab  
3. Switch to Balance Graph tab to see trends  

Tech
- HTML5 | CSS3 | JavaScript | Chart.js | localStorage
